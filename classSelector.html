  1. Selectors in CSS are used to "find" (or select) the HTML elements you want to style.

  2. We can divide the selectors in CSS into five categories:

    • Universal Selector
    • Type Selector
    • Class Selector
    • ID Selector
    • Group Selector
Example:-

CSS Selectors

This paragraph is styled using a class selector.

This paragraph is styled using an ID selector.

This paragraph is styled using a type selector.

Class Selector:- A class selector is used to select HTML elements with a specific class attribute. It is defined by a period (.) followed by the class name.

ID Selector:- An ID selector is used to select a single HTML element with a specific id

Id selector is defined by a hash (#) followed by the id name.

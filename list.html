Details of List Tags

  • Unordered List
  • Ordered List


Details of Unordered List:-

  • The ul tag is used to create an unordered list.
  • Items in an unordered list are typically displayed with bullet points.

Details of Ordered List:-
  1. The ol tag is used to create an ordered list.
  2. Items in an ordered list are typically displayed with numbers or letters.

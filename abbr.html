Abbreviation Tag in HTML

The <abbr> tag in HTML is used to define an abbreviation or acronym. When you hover over the abbreviation, a tooltip appears showing the full form of the abbreviation.

Example:-

  • The HTML is the standard markup language for creating web pages.
  • The CSS is used for styling web pages.
  • The JSON is a lightweight data interchange format.
